# Plan de escalada · León

Plan de entrenamiento de escalada (V7→V10 boulder · 7a→8a deportiva), interactivo.
Versión web: ver **GitHub Pages** de este repo.

Marca sesiones, registra cargas del loading pin/lastre y navega las 16 semanas en 4 fases.
Tu progreso se guarda local en tu navegador (localStorage).

Generado con investigación verificada (Eva López, Tyler Nelson, Yves Gravelle, Lattice).
